## 🚀 Getting Started

1. Navigate to any project directory
2. Open the Jupyter notebook (`.ipynb` file)
3. Install required dependencies
4. Run cells sequentially to reproduce the analysis

### Example: Running a Project

```bash
# Navigate to a project
cd E-commerce-JD_Comment_Sentiment_Analysis

# Install dependencies
pip install pandas numpy jieba matplotlib wordcloud

# Open Jupyter notebook
jupyter notebook jd_comment_sentiment_analysis.ipynb
```
