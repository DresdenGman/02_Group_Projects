# Group Projects — Python Data Analysis Portfolio

> **17 Projects · 8 Domains · Collaborative & Teaching Work**

A diverse collection of group projects, teaching examples, and collaborative research covering e-commerce, social science, education, real estate, tourism, workplace analytics, and more. Each project applies practical data science techniques to real-world problems.

---

## 🎯 Highlights

- **NLP Sentiment Analysis** — Chinese text processing with Jieba segmentation and Naive Bayes classification on real Weibo and JD.com data
- **National Census Visualization** — Comprehensive demographic analysis across multiple dimensions using Pyecharts
- **50 Pyecharts Examples** — A complete teaching collection demonstrating interactive visualization techniques
- **Automated Reporting** — Supermarket sales data pipeline with automated pivot analysis
