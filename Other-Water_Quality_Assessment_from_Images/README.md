# Water Quality Assessment from Images

## Project Overview

This project uses computer vision and machine learning techniques to assess water quality from images. The project demonstrates image classification, feature extraction, and SVM modeling for environmental monitoring applications.

## Objectives

1. Extract features from water quality images
2. Build classification models to assess water quality
3. Evaluate model performance
4. Provide automated water quality assessment

## Methodology

1. **Data Exploration**: Analyze image data characteristics
2. **Feature Extraction**: Extract relevant features from images
3. **Model Building**: Build Support Vector Machine (SVM) model
4. **Model Evaluation**: Assess model performance and accuracy

## Key Features

- Computer vision application
- Image classification
- Environmental monitoring
- Automated quality assessment

## Technologies Used

- OpenCV, PIL - Image processing
- pandas, numpy - Data processing
- scikit-learn - SVM classification
- Feature extraction techniques

## Project Structure

- `water_quality_assessment_from_images.ipynb` - Main analysis notebook
- `water_quality_assessment_from_images.html` - Exported visualization
- `svcmodel.pkl` - Trained SVM model
- `data/` - Image dataset folder (203+ images)
- `PROJECT_DESCRIPTION.txt` - Detailed project description

## Usage

1. Open the Jupyter notebook (`.ipynb` file)
2. Install required dependencies: `pip install pandas numpy opencv-python pillow scikit-learn`
3. Run cells sequentially to reproduce the analysis

## Results

The project provides:
- Automated water quality assessment from images
- Classification model for quality detection
- Environmental monitoring capabilities
