# JD Comment Sentiment Analysis

## Project Overview

This project performs sentiment analysis on JD (Jingdong) e-commerce platform customer reviews using natural language processing techniques. The project demonstrates NLP skills, text mining, and customer feedback analysis.

## Objectives

1. Analyze sentiment of customer reviews
2. Extract key insights from customer feedback
3. Create word clouds for positive and negative reviews
4. Provide insights for product improvement and customer service

## Methodology

1. **Data Preprocessing**: Clean and preprocess review text
2. **Text Processing**: Chinese word segmentation using jieba
3. **Sentiment Analysis**: Classify reviews as positive or negative
4. **Word Cloud Generation**: Visualize most common words in positive and negative reviews
5. **Analysis**: Extract insights and patterns

## Key Features

- NLP sentiment analysis on e-commerce reviews
- Word cloud visualization
- Customer feedback analysis
- Chinese text processing

## Technologies Used

- pandas, numpy - Data processing
- jieba - Chinese word segmentation
- wordcloud - Word cloud generation
- matplotlib - Visualization

## Project Structure

- `jd_comment_sentiment_analysis.ipynb` - Main analysis notebook
- `jd_comment_sentiment_analysis.py` - Python script version
- `jd_comment_sentiment_analysis.html` - Exported visualization
- `京东评论数据.csv` - Review dataset
- `词云图.png`, `消极评论词云.png` - Word cloud visualizations
- `SimHei.ttf` - Chinese font file
- `PROJECT_DESCRIPTION.txt` - Detailed project description

## Usage

1. Open the Jupyter notebook (`.ipynb` file)
2. Install required dependencies: `pip install pandas numpy jieba wordcloud matplotlib`
3. Run cells sequentially to reproduce the analysis

## Results

The project provides:
- Sentiment classification of customer reviews
- Word cloud visualizations
- Customer feedback insights
- Product improvement recommendations
