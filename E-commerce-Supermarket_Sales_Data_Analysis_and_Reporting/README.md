# Supermarket Sales Data Analysis and Reporting

## Project Overview

This project performs comprehensive analysis of supermarket sales data and automates report generation. The project demonstrates data pivot analysis, automated reporting, and business intelligence capabilities.

## Objectives

1. Analyze supermarket sales trends and patterns
2. Perform data pivot analysis
3. Automate sales reporting
4. Provide business insights for inventory and sales management

## Methodology

### Analysis Structure (based on outline):
1. **Introduction**: Purpose and data sources
2. **Overall Data Overview**: Dataset description, cleaning, and EDA
3. **Sales Trend Analysis**: Monthly and weekly sales changes, time trends
4. **Product Sales Analysis**: Best-selling categories, brands, seasonal patterns
5. **Customer Analysis**: New vs returning customers, regional distribution, demographics
6. **Store Analysis**: Sales comparison, product analysis, customer sources by store
7. **Conclusions and Recommendations**: Summary and improvement measures

## Key Features

- Automated data pivot analysis
- Sales reporting automation
- Business intelligence dashboards
- Comprehensive sales analysis

## Technologies Used

- pandas, numpy - Data processing
- openpyxl, xlrd - Excel automation
- matplotlib, seaborn - Visualization
- Automated report generation

## Project Structure

- `supermarket_sales_data_analysis.ipynb` - Main analysis notebook
- `supermarket_sales_automated_reporting.ipynb` - Automated reporting notebook
- `supermarket_sales_data_analysis.html` - Exported visualization
- `supermarket_sales_automated_reporting.html` - Automated report
- `order.csv`, `order-14.1.csv`, `order-14.3.csv` - Sales datasets
- `ChatGPT-Data_Analysis_Outline.txt` - Analysis outline (English)
- `PROJECT_DESCRIPTION.txt` - Detailed project description

## Usage

1. Open the Jupyter notebook (`.ipynb` file)
2. Install required dependencies: `pip install pandas numpy matplotlib seaborn openpyxl`
3. Run cells sequentially to reproduce the analysis
